Download and Install instructions for Ubuntu

In order to download and install Bang do the following (copy and paste in the terminal): 
	###########
# Install dependencies:
sudo apt-get -y install libglew-dev qt5-default libfreetype6-dev libopenal-dev libalut-dev libassimp-dev qtcreator

# Download and Install
git clone https://github.com/Bang3DEngine/Bang  # Clone Bang git repo
cd Bang
./scripts/fullCompile.sh EDITOR RELEASE_MODE # Compile the editor
./scripts/fullCompile.sh GAME   RELEASE_MODE # Compile the game base
###########
